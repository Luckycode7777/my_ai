# Тренажёр Арифметики

Веб-приложение для тренировки сложения и вычитания в уме.

## Возможности

- **Выбор операции**: сложение или вычитание
- **Режимы показа ответа**:
  - Ручной: кнопка показывает ответ, затем следующий пример
  - Автоматический: с настраиваемым таймером (1-60 сек)
- **Адаптивный дизайн**: работает на web, Android, iOS
- **Приятный интерфейс**: светло-зелёный фон, округлые кнопки

## Запуск

Откройте `index.html` в браузере или разверните на веб-сервере.

Для установки как PWA на мобильные устройства:
1. Откройте в браузере (Chrome/Safari)
2. Выберите "Добавить на главный экран"
3. Приложение установится как нативное

## Технологии

- HTML5
- CSS3 (градиенты, адаптивность)
- Vanilla JavaScript
- PWA (Progressive Web App)
