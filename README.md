Неделя 1 — Работа с текстом

Языковая модель = работа с текстом + числами.

Мини-проект №1: Анализ текста
Задача:

Создай файл text.txt
Вставь туда любой текст (можно 1–2 страницы книги или статьи)

Напиши программу, которая:
- читает файл
- переводит текст в нижний регистр
- считает частоту каждого символа
- выводит 10 самых частых символов и их количество
